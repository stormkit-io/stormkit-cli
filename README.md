# stormcli

Is a tool for manage [Stormkit.io](https://stormkit.io) via CLI.

## commands

### apps

list 

```
$ stormcli apps
github/stormkit-dev/sample-project
github/giuliobosco/web-project
```


## configuration

File `~/stormcli.yaml`:

```
app:
  bearer_token: "token" #bearer authentication token
```
